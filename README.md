# **TestCoCa**

Executes a program for each test in its test suite to measure certain
quantities, like branch coverage.

To use --testcomp mode, label-adder bibary built from https://gitlab.com/sosy-lab/software/test-suite-validator/-/tree/main/label-adder
must be present in the same directory as TestCoCa.py.
